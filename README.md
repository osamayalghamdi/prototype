# Midan: Your AI-Powered Sporting Event Assistant

## Project Overview

Midan is an innovative AI-powered assistant designed to enhance the experience of attending sporting events. It leverages a Retrieval-Augmented Generation (RAG) workflow, built using the Dify platform locally, to provide real-time information, navigation assistance, and personalized recommendations, ensuring fans have the best possible experience at any stadium.

## Recognition

We're proud to announce that Midan has qualified for the finals of the AI League competition, organized by the Saudi Center for Artificial Intelligence (SCAI), selected from over 600 teams representing 20 countries worldwide. This achievement highlights our commitment to innovative AI solutions in the sports and entertainment sector.

## Key Features

- **Real-Time Updates**: Get live scores, player stats, and event highlights directly on your device.
- **Stadium Navigation**: Easily find your way around the stadium with interactive maps and directions to your seat, food stalls, restrooms, and exits.
- **Personalized Recommendations**: Receive tailored suggestions for food, merchandise, and activities based on your preferences.
- **Emergency Assistance**: Access emergency contacts and first aid locations quickly in case of an emergency.
- **Customizable Experience**: Upload your own stadium layout for a personalized navigation experience.
- **Multilingual Support**: Full support for all languages, essential for international events including the upcoming World Cup and other international tournaments in Saudi Arabia.
- **Voice Input**: Record voice messages instead of typing for a more convenient interaction experience.
- **File Attachments**: Upload documents like tickets for quick access and information retrieval.

## AI Workflow

Midan uses a sophisticated Retrieval-Augmented Generation (RAG) workflow to provide accurate and real-time information to users:

- **Multiple Language Models**: Integrates Qwen 3, Ollama3.2, and GPT-4o models working together to ensure optimal performance and response quality.
- **Custom Search Agent**: Features a dedicated search agent that combines search results with stored data in real-time, ensuring information is always up-to-date.
- **Dynamic Knowledge Integration**: Continuously updates information based on user interactions and external data sources.

## Getting Started

To get started with the Midan frontend, follow these steps:

```sh
# Step 1: Clone the repository using the project's Git URL.
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory.
cd midan # Or your project directory name

# Step 3: Install the necessary dependencies.
npm install

# Step 4: Start the development server.
npm run dev
```
